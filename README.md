# ApiWithChart

Приложение погоды с Chart.js:

Поробовать htpps://foamy-mark.surge.sh/  (Weather-lstu-with-chart.js)

- HTML/CSS

- JavaScript

- Chart.js

- Axios

- Siema

- Webpack

установка проекта на пк для разработки:

1. Клонируем репозиторий и переходим в папку.

2. Устанавливаем зависимости.

```shell
npm install
```

3. Запускаем.

```shell
npm start
```
